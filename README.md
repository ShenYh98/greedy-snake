# zspaceMQ
## 引言
这是一款godot开发的贪吃蛇小游戏，我边学边做自娱自乐的小demo，里面将godot主要的一些概念、控件、接口等都基本涉及，感兴趣的读者可自行拉源码阅读学习，觉得不错麻烦点个星星哦❤，后续我会持续优化更新，感谢支持！
